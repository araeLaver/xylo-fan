# XYLO 프로젝트 WBS (Work Breakdown Structure)

## 문서 정보
- **작성일**: 2025-01-11
- **프로젝트**: XYLO Web3 크리에이터 플랫폼
- **목적**: 기능별 작업 분해 구조 (Work Breakdown Structure)
- **범위**: MVP 전체 기능 (백엔드 + 프론트엔드)

---

## 1. 프로젝트 개요

### 1.1 프로젝트 목표
- YouTube Shorts 크리에이터를 위한 Web3 리워드 플랫폼 구축
- 포인트 기반 리더보드 시스템 및 NFT 보상 제공
- MVP 기간 내 XLT 토큰 교환 시스템 구현

### 1.2 주요 이해관계자
- 개발팀: 백엔드 1명, 프론트엔드 1명
- 디자인팀: UI/UX 디자이너 1명
- QA팀: 테스터 1명

### 1.3 프로젝트 일정
- **총 개발 기간**: 12주 (디자인 완료 후 시작)
- **백엔드 잔여 작업**: 11일
- **프론트엔드 작업**: 60일 (디자인 완료 후)
